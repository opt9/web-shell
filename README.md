# web-shell
**tags:** java, spring-boot, gradle, swagger, unix, bash, shell, terminal, command line

**how build:**
```
gradle clean build
```

**how run:**
```
java -jar ./build/libs/web-shell.jar
```

**example how use:**
open in browser
```
http://localhost:8080/shell/uname -a
```

**supply:**
all unix

**tested:**
OS X 10.11 El Capitan, Ubuntu 16.04 LTS
